# Purpose of the Outer Rim Tournaments 

The main purpose of the Outer Rim Tournaments is to provide a platform for players to participate in competitive Star Wars Unlimited namely Twin Suns tournaments. The application allows users to register, create and join tournaments, and track their progress and rankings.

# Features

- User registration and authentication
- Tournament creation and management
- Joining and participating in tournaments
- Tracking progress and rankings

# Technologies Used

- Frontend: React, Redux, React Router
- Backend: Node.js, Express
